## Installation

1. In your Uptime Kuma dashboard, go to your status page.
2. Click **Edit Status Page**.
3. Scroll down to the **Custom CSS** field.
4. Copy the contents of **[mini-kuma.css](mini-kuma.css)** (MiniKuma theme) and paste it into that field.
5. Click **Save** to apply.


### Features

- Minimal and modern design for clean status pages
- Full support for Uptime Kuma both v1 and v2
- Light theme friendly
- Hidden refresh timer for distraction-free monitoring (optional to restore)
- Simple and well-structured CSS for easy customization


### Personalization

You can tweak MiniKuma by editing the variables within the `:root {}` block inside the CSS file.
